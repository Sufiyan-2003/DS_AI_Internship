How It Works

This calculator demonstrates floats and division in programming:

share_per_person = total_bill / num_people
1 Total bill is stored as a float to handle decimals
2 Number of people is stored as an int (whole number)
3 Division returns a float result, even when dividing by an integer

Python Code Example

# Get inputs and convert to appropriate types
total_bill = float(input("Enter total bill: "))
num_people = int(input("Enter number of people: "))

# Calculate share (division operation)
share_per_person = total_bill / num_people

# Print result
print(f"Each person pays: ₹{share_per_person:.2f}")
